# Deloitte Cybersecurity Virtual Experience – Forage

## Overview

This project documents my participation in the Deloitte Cybersecurity Virtual Experience Program hosted on the Forage platform.

The simulation focuses on practical cybersecurity tasks performed by analysts in a Security Operations Center (SOC).

During the simulation, I investigated security logs, identified suspicious activity, and analyzed potential cyber threats.

## Tasks Completed

* Investigated suspicious login activity  
* Analyzed system logs and authentication events  
* Identified potential indicators of compromise  
* Applied threat detection and incident analysis techniques
